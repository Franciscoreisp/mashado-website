# Mashado

Mashado is a rebranding and redesign project based on a previous community website, with improvements in layout, structure, and overall user experience.

## About the project
This project was developed as part of my portfolio to showcase website improvement, redesign, and adaptation skills.  
The main goal was to transform an existing web platform into a new concept called **Mashado**, with a refreshed identity and a more organized user experience.

## Main improvements
- Rebranding of the original website into Mashado
- Improved visual structure and layout
- Better organization of content
- Enhanced overall user experience
- Adaptation of the platform to a new identity

## Technologies used
- HTML
- CSS
- JavaScript

## Purpose
This project was created to demonstrate my ability to analyze an existing website, redesign its structure, and adapt it into a new concept with a different identity.
